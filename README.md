A. Manual Testing:

Testcase nào bug thì tô màu đỏ.<br>
( Lưu ý:  tạo 3 sheet riêng cho từng level.) <br>
Do số lượng testcase khá nhiều nên đối với phần Manual testing các thành viên phải phân chia nhau để viết testcase. <br>
